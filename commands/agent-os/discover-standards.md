# /discover-standards — Extract patterns from your codebase

## Description
Analyze your codebase to extract coding patterns, conventions, and best practices into documented standards.

## Usage
Run this command when you want to:
- Establish coding standards for a new project
- Document existing patterns in an inherited codebase
- Update standards after a major refactor

## Process
1. Scan the codebase structure and key files
2. Identify naming conventions, patterns, and architecture decisions
3. Document findings in agent-os/standards/
4. Update the standards index

## Output
- `agent-os/standards/coding-standards.md` — Documented conventions
- `agent-os/standards/index.yml` — Updated index of all standards
