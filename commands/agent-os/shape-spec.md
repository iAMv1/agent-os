# /shape-spec — Create better specifications

## Description
Transform a rough idea into a structured, actionable specification that leads to better builds.

## Usage
Run this command when you want to:
- Plan a new feature before implementation
- Clarify requirements before coding
- Create a spec for the agent to follow

## Process
1. Understand the goal and constraints
2. Define success criteria and edge cases
3. Structure the spec with clear sections
4. Identify risks and mitigation strategies
5. Create implementation phases

## Output
- Structured specification document
- Implementation phases with acceptance criteria
- Risk assessment and mitigation plan
