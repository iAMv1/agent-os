# /ralph-loop — Start iterative self-improvement cycle

## Description
Begin a Ralph Loop cycle: Plan → Execute → Review → Learn → Improve → Repeat.

## Usage
Run this command when you want to:
- Start a self-improvement cycle for a feature
- Review and improve existing code
- Learn from mistakes and update approach

## Process
1. **Plan**: Define what we're building and how we'll know it's good
2. **Execute**: Build using the best available approach
3. **Review**: Critically evaluate against standards
4. **Learn**: Extract lessons from what worked and what didn't
5. **Improve**: Update approach based on lessons learned
6. **Repeat**: Start next cycle with improved knowledge

## Output
- Improved implementation
- Documented lessons learned
- Updated approach for next cycle
